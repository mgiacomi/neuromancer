Day 1: "The Pure Math" (NumPy only). Spend one hour building a single neuron using only numpy. No AI frameworks. This will show him how Matrix Multiplication and a simple Derivative work in code.

Day 2-5: "The Framework" (PyTorch). Move the project into PyTorch. You'll get the power of the GPU and pre-built layers, but you'll still be "driving the car" yourself.

The Goal: Build a "Digit Classifier" (MNIST). It’s the "Hello World" of AI and provides a very satisfying visual result when you draw a number and the math correctly identifies it.
